# Resource 目录保护规则

本文件继承根级 [AGENTS.md](../AGENTS.md)。`Resource/` 承载共享配置、图标和静态资源，资源名称、路径、URI、格式和复制行为均可能是消费者契约。

- 修改前搜索代码、XAML、项目复制项、配置加载和发布脚本中的全部消费者；禁止只凭文件名判断未使用。
- 重命名、移动、删除、格式/默认值或复制策略变化必须验证兼容；共享或运行配置变化至少 R2，发布配置按 R3。
- 图片、日志样例和配置不得包含凭据、客户/生产数据或可识别设备信息；第三方素材记录来源和许可证。
- 验证资源解析、主题/缩放（适用时）、缺失/损坏行为、消费者构建和运行时复制；回滚必须恢复权威资源及引用。

影响调查见 [变更影响地图](../docs/development/change-impact-map.md)。
