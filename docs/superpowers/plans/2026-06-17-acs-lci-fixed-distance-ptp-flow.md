# ACS LCI Fixed Distance PTP Flow Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Change `BuildLciFixedDistancePulseXsegScript` so it generates the ACSPL+ fixed-distance pulse flow from the confirmed reference script: enable axes, move to start, initialize LCI, configure `FixedDistPulse`, enable laser, and run a `PTP/e` move to the stop point.

**Architecture:** Keep the existing parameter model and UI surface. Use `param.Points[0]` as `XStartPos/YStartPos` and `param.Points[^1]` as `XStopPos/YStopPos`, preserving `StartDistance` and `EndDistance` as pulse start/end distances. Only the generated ACSPL+ script and its regression test change.

**Tech Stack:** C# .NET 8 WPF project, ACSPL+ script generation, custom console test project.

```csharp
private static void AppendAxisMotionTuning(StringBuilder builder, string axisName)
{
    builder.AppendLine($"VEL({axisName}) = 10");
    builder.AppendLine($"ACC({axisName}) = 10 * VEL({axisName})");
    builder.AppendLine($"DEC({axisName}) = 10 * VEL({axisName})");
    builder.AppendLine($"KDEC({axisName}) = 10 * ACC({axisName})");
    builder.AppendLine($"JERK({axisName}) = 10 * ACC({axisName})");
}
```
